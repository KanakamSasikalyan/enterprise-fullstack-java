## Java Inner Class

- In java, we can create the nested classes or inner class.
- If we create a static class as as the inner class it do not new a reference of the outer class, the object of the inner class is enough.
- The Inheritence is possible with in the innerclasses. ( expermentally proved).

## Final Keyword

We can use the final keyword in many differnt scenarios to make it immutable nature of class , objects, variables (constants), and methods.
1. final class
2. final variables
3. final methods
4. final objects
### KeyPoints of final keyword
1. Final class cannot be inherited
2. Final variables cannot be changed once it assigned a value.
3. Final variables are can be  declared, initialized with help of the constructor.
4. Final methods cannot be overridden from the parent class.
