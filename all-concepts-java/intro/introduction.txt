Features of Java:

- OOPS: java revolves around the concept of objects, which has been built on the solid principles like abstraction,
        inheritance, encapsulation, polymorphism
- Simple: It is easy to learn just like c/c++
- Platform independent: java byte code (generated by javac compiler) makes the java platform independent
                        it could be run on any runtime environment i.e. operating system with compatible JVM inside it.

- Robust: Java is strong programming language, it doesn't have complex concepts like pointers, operator overloading or
          memory management concepts.
- Multithreading: Java is supports multithreading, multithreading is the concept of concurrent execution of multiple
threads at a time to utilize the maximum CPU i.e to optimize the performance of the current process.

Other features include: Distributed, Automatic Memory Management, Garbage Collection etc...

JVM vs JRE vs JDK

JVM : is an abstract machine which provides the runtime environment for the bytecode to execute it.
      it can also contribute memory management, security and garbage collection

JRE : provides the runtime env for the java applications, it includes JVM and rt (runtime) libraries used by the
      JVM to execute the bytecode.
JDK : It is the complete software package it includes JRE + Compilers , Debuggers and other tools
